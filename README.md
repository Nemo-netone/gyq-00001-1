# 📋 网页剪贴板工具

一个简单易用的网页版剪贴板工具，帮助你保存和管理复制过的文字内容，再也不怕重要文字被覆盖丢失！

## ✨ 功能特性

- **文字保存**：粘贴文字即可保存，支持 Ctrl+Enter 快捷键
- **列表展示**：所有保存的内容按时间倒序排列展示
- **一键复制**：点击按钮即可将内容复制到系统剪贴板
- **删除功能**：不需要的内容可以随时删除
- **上下移动**：像电脑剪贴板一样调整内容顺序
- **键盘操作**：支持方向键选择、Ctrl+C 复制、Delete 删除等快捷键
- **本地存储**：使用 SQLite 数据库，数据保存在本地

## 🚀 快速开始

### 环境要求

- Node.js >= 14.0.0

### 安装依赖

```bash
npm install
```

### 启动服务

```bash
npm start
```

启动后，在浏览器中访问：<http://localhost:3000>

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Enter` | 在输入框中保存内容 |
| `↑ / ↓` | 在列表中上下选择 |
| `Ctrl + C` | 复制当前选中的内容 |
| `Delete` | 删除当前选中的内容 |

## 🛠 技术栈

- **后端**：Node.js + Express
- **数据库**：SQLite (sql.js，纯 JS 实现，无需编译)
- **前端**：原生 HTML/CSS/JavaScript
